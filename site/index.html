    Welcome to MkDocs¶

    For full documentation visit mkdocs.org.

    Commands¶

    • mkdocs new [dir-name] - Create a new project.
    • mkdocs serve - Start the live-reloading docs server.
    • mkdocs build - Build the documentation site.
    • mkdocs -h - Print help message and exit.

    Here‚Äôs a MkDocs configuration cheat sheet to help you quickly set up and customize your documentation project using the mkdocs.yml config file.

site_description: Documentation for my AI/ML learning

  - Home: index.md
  - About: about.md
  - ML:
      - Basics: ml/basics.md
      - Models: ml/models.md
  - AI:
      - Concepts: ai/concepts.md

  name: material  # Recommended theme
  palette:
    primary: blue
    accent: indigo
  font:
    text: Roboto
    code: Roboto Mono

docs_dir: docs
site_dir: site
